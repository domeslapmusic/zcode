This is ZCode's Hello World application. Please go there for some documentation or just browse the repository if you fancy.

If you're reading this you either don't have the right Flash player version or your browser doesn't support JavaScript.

To compile this application:

  • get a copy of the source
  • if you're using MTASC, use something like this:
    • mtasc -swf "helloworld.swf" -header 500:500:21 -version 8 -cp "/path/to/your/classes" HelloModel.as -main -trace tv.zarate.utils.Trace.trc
  • if you're using Flash IDE:
    • add to your classpath the folder in which you've downloaded the code. Usually you do that in AS2 settings dialog
    • in the first frame add: import tv.zarate.projects.demo.zcode.HelloModel;
    • then: HelloModel.main(this);
