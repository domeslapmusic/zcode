Hay 3 mini-aplicaciones en este código fuente. Para compilar cada una de ellas simplemente copia y paga el código de la que quieras en el primer frame de tu película.

- Simple Dispatcher:

import tv.zarate.Projects.loqueyosede.simpleReceiver;
simpleReceiver.main(this);

- Dispatcher:

import tv.zarate.Projects.loqueyosede.receiver;
receiver.main(this);

- MVC:

import tv.zarate.Projects.loqueyosede.MVCApplication;
MVCApplication.main(this);

Para compilar desde MTASC utilizar el método -main con las clases anteriores.
