Autobackup options

A few words about autobackup. The idea is:

  • Everytime the *admin* enters ZBooks, the date of the last backup is checked. If is time to backup, the process starts.
  • A new file is created depending on your backup format settings.
  • That file could be either send to the admin via email or stored in the server. Do NOT store the file in the server if you have the ZBooks' data folder inside the public root of your server, otherwise the file will be accessible (including user names and emails) by anyone.
